常用自定义样式

本文介绍一些常用的自定义样式。您可以将代码粘贴到脚本“外观样式”选项卡的自定义 CSS 的文本框中定制网页样式，也可以使用 Stylish 等扩展添加自定义样式。部分样式需要安装了 YAWF 脚本才能生效。

目录

隐藏搜索框

请参阅：添加新浪微博的搜索到您的浏览器搜索框中

.WB_global_nav .gn_search { display: none !important; }

如要隐藏“热搜词”，建议使用 Adblock Plus 等扩展，并添加如下规则

http://s.weibo.com/ajax/jsonp/suggestion?*

隐藏左上角微博图标

.gn_logo { display: none; }

隐藏微博发布框

隐藏后可以使用右上角的快速发布。

#v6_pl_content_publishertop { display: none !important; }

隐藏右下返回顶部按钮

#base_scrollToTop { display: none !important }

调整合并两侧栏后导航栏宽度

如果导航栏屏蔽掉了“应用”和“游戏”的链接，或者隐藏了搜索框，可以使用下面这段 CSS 使顶栏变窄：

.WB_global_nav .gn_search .W_input { max-width: 200px; } body[yawf-merge-left] .gn_header { width: 840px !important; } body[yawf-merge-left="left"] .gn_logo { margin-right: 95px !important; }

调整滚动条宽度

这里的滚动条指微博自己的界面中绘制的滚动条，左栏分组列表过长时的滚动条。并非由系统样式决定的滚动条。如果你觉得那个滚动条太窄，可以应用以下样式将其加宽。调整下例样式中的8px可以调整宽度，数值越大则越宽。

.W_scroll_y { width: 8px !important; } .W_scroll_y .bar { border-radius: calc(8px / 2) !important; }

自定义折叠微博样式

可以通过对[node-type="feed_list"] .WB_feed_type[yawf-display$="-fold"]::before、[node-type="feed_list"] .WB_feed_type[yawf-display$="-fold"]:hover::before添加样式自定义折叠提示的样式，例如：

[node-type="feed_list"] .WB_feed_type[yawf-display$="-fold"]::before { color: #fff; background: #522; border-color: #777; } [node-type="feed_list"] .WB_feed_type[yawf-display$="-fold"]:hover::before { color: #fff; background: #351; border-color: #777; }

加宽首页微博

以下样式可以加宽首页微博列表（阅读模式除外），您需要做一些修改：

  • Firefox 31 及更新版本的用户，您只需要修改第一行的 750 为你喜欢的宽度即可，不加宽时的宽度为 600 。
  • Firefox 30 以下，或其他浏览器的用户， 您需要查找替换下文中所有的 var(--yawf-main-width) 为您想要的宽度（如750px）。
:root { --yawf-main-width: 750px; } .WB_frame { width: calc(var(--yawf-main-width) + 400px); } body[yawf-merge-left]:not([yawf-weibo-only]) .WB_main .WB_frame { width: calc(var(--yawf-main-width) + 240px) !important; } .WB_frame { width: calc(var(--yawf-main-width) + 400px); } body[yawf-merge-left]:not([yawf-weibo-only]) .WB_main .WB_frame { width: calc(var(--yawf-main-width) + 240px) !important; } #plc_main { width: calc(var(--yawf-main-width) + 20px + 230px) !important; } .WB_frame .WB_main_c { width: var(--yawf-main-width); } .WB_frame .WB_main_c .WB_tab_a .tab_box_a .fr_box { width: calc(var(--yawf-main-width) - 300px); } a.W_gotop { margin-left: calc(var(--yawf-main-width) / 2 + 200px); } body[yawf-merge-left]:not([yawf-weibo-only]) a.W_gotop { margin-left: calc(var(--yawf-main-width) / 2 + 130px) }

隐藏漏斗图标

.gn_search .W_input { box-sizing: inherit !important; height: 20px !important; } .yawf-gn_set_list { display: none !important; }

隐藏后还可以通过旁边齿轮图标菜单中的“过滤器设置”项目打开设置窗口。

此外，如果您因为自定义 CSS 无法打开设置对话框，请参考常见问题中的相关说明进行处理。
