看真正想看的微博 (YAWF)

安装脚本

新浪微博根据关键词、作者、话题、来源等过滤微博；修改版面。

脚本尚处于开发之中，功能尚不稳定，不能保证更新后可以保留原有设置。

请使用 Greasemonkey 安装本脚本。

当前功能

  • 根据关键字、正则式、作者、来源等过滤微博
  • 过滤推广微博等混入微博列表的东西
  • 其他一些广告过滤
  • 清理微博版面
  • 添加查看大图链接

关于

YAWF (Yet Another Weibo Filter) 使用 MIT 协议授权。
您可以访问脚本的主页 https://github.com/tiansh/yawf 获取详细信息。
本脚本参考并使用了眼不见心不烦脚本的部分代码。

Yet Another Weibo Filter

Install Script

filter Sina Weibo by keywords, original, topic, source, etc.; modify layout

script have not been done. It may not stable. And you may lose your setting after update.

Please use Greasemonkey to install this script

Current Function

  • Filter weibo by keyword, regexp, author, source, etc.
  • Filter out suggestion Weibo and other contents in Weibo list
  • Ad Block
  • Clean webpage layout
  • Add "View Original" link
